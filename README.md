# churn-prediction
This project predicts whether a customer is likely to churn (leave the company) or not based on historical customer data. It uses data preprocessing techniques, handles imbalanced data using SMOTE, and applies a Random Forest Classifier to build the prediction model. The model achieves around 83% accuracy.
📊 Customer Churn Prediction
📌 Overview

This project predicts whether a customer will churn or not using machine learning based on historical customer data.

🎯 Objective

To identify customers likely to leave and help businesses improve retention.

🧠 Approach
Data preprocessing
Encoding categorical features
Handling imbalance using SMOTE
Model: Random Forest Classifier
Evaluation using confusion matrix & metrics
🛠️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn
Imbalanced-learn
📊 Results
Accuracy: ~83%
Improved churn detection using SMOTE
Balanced model performance
